# SilentSuite Bridge

Local E2EE CalDAV/CardDAV sync daemon for SilentSuite.

Connects to `server.silentsuite.io` via the Etebase protocol, decrypts/encrypts data locally, and exposes CalDAV/CardDAV endpoints on `localhost:37358` for use with any standard PIM client (Thunderbird, Apple Calendar, GNOME Calendar, Evolution, etc.).
